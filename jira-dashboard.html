🔧 Setup Instructions

Step 1: Create an OAuth 2.0 App

  1. Go to Atlassian Developer Console
  2. Click "Create app" and select "OAuth 2.0 (3LO)"
  3. Name your app and accept the terms
  4. Click "Create"

Step 2: Configure OAuth Redirect URI

  1. In the app settings, go to "Authorization"
  2. Add a Redirect URI:
    Loading...
  3. Save your changes

Step 3: Configure Your App

  1. In the app settings, go to "Permissions" and enable: "read:jira-work", "read:me", "offline_access"
  2. Copy your Client ID and Client Secret
  3. Update the JavaScript variables below:
// Update these with your OAuth credentials
const CLIENT_ID = 'YOUR_CLIENT_ID';
const CLIENT_SECRET = 'YOUR_CLIENT_SECRET';
const REDIRECT_URI = window.location.origin + '/callback';
const AUTH_URL = 'https://auth.atlassian.com/authorize';
const TOKEN_URL = 'https://auth.atlassian.com/oauth/token';
